# C and C++ scripts  

### Compile C with:  
Where `*.c` is the name of the C file that you are trying to compile. `e` is the name of the executable, which you could also change to any name you want.  

```
clang *.c -o e 
gcc *.c -o e
```  

### Run with:  

```
./e
```

**Alternatively, use a makefile**
