# Reading CSV files with Eigen and Boost  

*The `Makefile` is using the `clang++` compiler. If you are using the `g++` compiler, make sure to change the `CXX` variable.*  

### Compile using **Make**  

```
make
```  

### Run with **Make**  

```
make run
```  

### Delete the `main` executable  

```
make clean
```
